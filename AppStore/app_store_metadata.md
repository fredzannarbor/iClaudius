## Description

iClaudius is the essential companion app for Claude Code users. View, understand, and manage all your Claude Code configuration in one beautiful native macOS interface.

**CONFIGURATION DASHBOARD**
See everything at a glance: your CLAUDE.md file hierarchy, custom slash commands, installed plugins, scheduled jobs, and account statistics. A health indicator instantly shows if your configuration has any issues.

**CLAUDE.MD HIERARCHY**
Understand how your configuration files stack up. View global (~/.claude/CLAUDE.md), user-level (~/CLAUDE.md), and project-specific configurations with their inheritance order clearly displayed.

**CUSTOM SLASH COMMANDS**
Browse all your custom commands and skills. Search instantly, see which ones are aliases, and open them for editing in your preferred editor.

**CRON JOB MANAGEMENT**
Full create, read, update, and delete support for scheduled automation. Use preset templates or create custom schedules for your Claude Code workflows.

**PLUGIN OVERVIEW**
See all installed plugins, which are enabled, and when they were installed.

**SMART SUGGESTIONS**
Get personalized recommendations based on your current setup. iClaudius analyzes your configuration and suggests complementary commands, automations, and optimizations.

**PRIVACY FIRST**
iClaudius runs entirely locally. No data collection, no network requests, no accounts required. Your configuration stays on your Mac.

**DESIGNED FOR CLAUDE CODE USERS**
Whether you're a power user with complex multi-project setups or just getting started with Claude Code customization, iClaudius helps you stay organized and discover new possibilities.
