# Privacy Policy for iClaudius

**Last Updated: December 27, 2024**

## Overview

iClaudius is a macOS application for viewing and managing Claude Code configuration files. This privacy policy explains how iClaudius handles your data.

## Data Collection

**iClaudius does NOT collect any data.**

Specifically, iClaudius:
- Does NOT collect personal information
- Does NOT collect usage analytics
- Does NOT collect crash reports
- Does NOT track user behavior
- Does NOT use cookies or similar technologies
- Does NOT share data with third parties

## Data Access

iClaudius accesses the following local files on your Mac to provide its functionality:
- `~/.claude/` directory and its contents
- `~/CLAUDE.md` and project-level CLAUDE.md files
- Your crontab (for viewing and editing scheduled jobs)

All data access is read-only except when you explicitly choose to:
- Add, edit, or delete cron jobs
- (All other configuration editing opens files in your default text editor)

## Network Access

**iClaudius does NOT make any network requests.**

The app runs entirely offline. It does not:
- Connect to the internet
- Send or receive data from any servers
- Require an internet connection to function

## Third-Party Services

iClaudius does not integrate with any third-party services, analytics platforms, or advertising networks.

## Data Storage

iClaudius does not store any data of its own. It only reads your existing Claude Code configuration files.

## Children's Privacy

iClaudius does not collect any information from anyone, including children under 13 years of age.

## Changes to This Policy

If we make changes to this privacy policy, we will update the "Last Updated" date at the top of this document.

## Contact

If you have questions about this privacy policy, please open an issue on our GitHub repository.

## Summary

**iClaudius is a privacy-first application that runs entirely locally on your Mac with zero data collection, zero network access, and zero tracking.**
